# Raycast DaVinci Resolve Commander

A Raycast extension that gives you instant, searchable access to DaVinci Resolve commands. Browse or fuzzy-search through 80+ commands organized by category, then hit Enter to send the shortcut directly to DaVinci Resolve.

## How It Works

1. Open Raycast and search for "DaVinci Resolve"
2. Browse or type to filter commands
3. Select a command and press Enter
4. The shortcut is sent to DaVinci Resolve via AppleScript

## Requirements

- macOS
- [Raycast](https://raycast.com)
- DaVinci Resolve (must be running)
- Accessibility permissions granted to Raycast
